## Project Overview

This project is called KitchenHub. It is a simple web application for managing recipes. I made this project for my friend with some guidance and help.

> Note: All features were implemented primarily by Sushita Roy Chowdhury, with guidance from external resourses.
> Database credentials used for testing belong to a friend, but the project logic and code are authored by Sushita Roy Chowdhury.

## Features
- Chef or admin can add, view, and manage recipes.
- Users can save and read recipes.
- Simple and user-friendly interface.

## Technologies Used

- **Frontend:** React + Tailwindcss

- **Backend:** Node.js + Express

- **Database:** MongoDB (Mongoose)


## How It Works:
1. Admins or Chefs can add new recipes with ingredients and instructions.
2. Admins or Chefs can edit or delete recipes with ingredients and instructions.
3. Users can view all recipes and save them.
